# Задачи

## 1. Обновить документацию

Добавить в документацию три новых блока из шаблона КП:

- **Описание услуг**
- **Условия оказания услуг**
- **Услуги должны соответствовать требованиям**

## 2. Создать требования для заполнения

Разработать требования, промпт и чек-лист для того, как агент будет заполнять эти три блока на основании информации от руководителя проекта

## 3. Переконфигурировать документацию

Обновить документацию в соответствии с реальным рабочим процессом, исходя из его опыта погружения в проект

## 4. Создать файл "Статус проекта"

Отразить в нём:

- Что уже сделано
- Какие блоки реализованы
- Риски
- Что осталось доделать
- Задачу для заказчика: согласовать требования к заполнению новых блоков документа

## 5. Создать Change Log

Описать, что именно изменилось в документации

## 6. Прислать обновлённую ссылку

Отправить ссылку на обновлённую документацию со всеми изменениями
